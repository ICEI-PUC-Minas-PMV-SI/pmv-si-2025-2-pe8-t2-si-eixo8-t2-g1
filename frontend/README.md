# Health Scheduler

## Project info

A health scheduling application built with React, TypeScript, and Vite.

## How to run locally

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <https://github.com/GabrielGNM/clinica-ana-eixo-8.git>

# Step 2: Navigate to the project directory
cd health-scheduler

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
